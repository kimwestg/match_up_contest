# 🫀 심장질환 주요 원인 분석 프로젝트  
**[2024년 산업맞춤 단기직무능력 인증 과정]**  
제2회 매치업 심화과정 경진대회 출품작  

---

## 📌 프로젝트 개요
이 프로젝트는 **로지스틱 회귀분석(Logistic Regression)** 과 **ANOVA(분산분석)**, **피처 셀렉션(Feature Selection)**, **랜덤 포레스트(Random Forest)** 를 활용하여 심장질환 발생에 영향을 미치는 주요 요인을 분석하고 예측 모델을 고도화한 결과물입니다.  
분석을 통해 **고위험군 선별**, **불필요한 검사·치료 최소화**, **공중보건 정책 설계** 등에 기여할 수 있는 인사이트를 도출했습니다.

---

## 🗂 데이터 정보
- **출처**: Heart Disease UCI (Kaggle)  
- **데이터 크기**: 304행 × 14컬럼 (결측치 없음, 클래스 비율 균형)  
- **변수 설명**:  
  - `age`: 나이  
  - `sex`: 성별 (1: 남성, 2: 여성)  
  - `cp`: 흉통 유형 (0: 전형 협심증, 1: 비전형 협심증, 2: 비협심증성 통증, 3: 무증상)  
  - `trestbps`: 병원 입원 시 혈압  
  - `chol`: 총 콜레스테롤 수치  
  - `fbs`: 공복 혈당 초과 여부  
  - `restecg`: 휴식 시 심전도  
  - `thalach`: 최대 심박수  
  - `exang`: 운동 유발 협심증  
  - `oldpeak`: ST 분절 차이  
  - `slope`: ST 분절 기울기 (0: 상승, 1: 평형, 2: 하강)  
  - `ca`: 주요 혈관 수 (0~4)  
  - `thal`: 핵의학 스캔 결과  
  - `target`: 심장질환 여부 (0: 없음, 1: 있음)  

---

## 🔎 분석 과정
1. **EDA**
   - 상관관계:  
     - 양의 상관관계 → `cp`(0.43), `thalach`(0.42), `slope`(0.35)  
     - 음의 상관관계 → `exang`(-0.44), `oldpeak`(-0.43), `ca`(-0.39)  
   - VIF ≒ 1 → 다중공선성 문제 없음  
2. **통계적 검정**
   - **ANOVA & Post-Hoc Test**  
     - target과 상관성이 높았던 **cp, slope, ca** 범주형 변수에서 높은 F값 및 범주 간 유의한 차이 확인  
     - 예: `cp=0`(전형 협심증), `slope=2`(하강형), `ca=0`(혈관 0개) 그룹의 발병률이 상대적으로 높음  
3. **피처 셀렉션**
   - 상관분석·ANOVA 결과 기반으로 중요 변수 선별  
   - 13 → 8개 변수로 축소하여 모델 단순화 및 해석력 강화  
4. **모델 학습 및 고도화**
   - 로지스틱 회귀분석 → 기본 모델 성능 확보  
   - 랜덤 포레스트 → 비선형 관계 반영 및 성능 개선 (GridSearchCV로 하이퍼파라미터 최적화)  
5. **성능 평가**
   - 교차검증(5-fold) 정확도: 77% ~ 90%, 평균 83.8%  
   - 최종 AUC: 93%  
   - Classification Report (테스트 세트):  
     | Class | Precision | Recall | F1-Score | Support |  
     |-------|-----------|--------|----------|---------|  
     | 0     | 0.89      | 0.86   | 0.88     | 29      |  
     | 1     | 0.88      | 0.91   | 0.89     | 32      |  
     - Accuracy: 85%
